# Firmware del nodo Heltec

Esta carpeta contiene los programas utilizados en el nodo Heltec ESP32-S3 con transceptor LoRa SX1262.

Se incluyen tanto pruebas iniciales de comunicación LoRa punto a punto como versiones orientadas a la transmisión LoRaWAN mediante The Things Network.

Por motivos de seguridad, las credenciales reales OTAA, como DEV_EUI, JOIN_EUI y APP_KEY, no se incluyen en los archivos publicados.

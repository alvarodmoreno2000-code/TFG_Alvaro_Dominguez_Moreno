# Validación preliminar mediante k-NN

Esta carpeta contiene el script y los resultados de la validación preliminar de las huellas radioeléctricas mediante un clasificador k-NN.

La validación se realizó utilizando las métricas RSSI, SNR y factor de expansión registradas durante la campaña experimental.

Los resultados incluidos permiten consultar las matrices de confusión, las predicciones obtenidas y los porcentajes de acierto por posición.

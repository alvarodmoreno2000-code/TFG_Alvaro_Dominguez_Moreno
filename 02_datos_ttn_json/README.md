# Datos originales exportados desde The Things Network

Esta carpeta contiene los archivos JSON originales exportados desde The Things Network durante la campaña experimental.

Los ficheros corresponden a las posiciones de medida H01, H02 y H03, registradas durante la validación del banco de pruebas LoRaWAN.

Estos datos fueron posteriormente convertidos a formato CSV y filtrados para conservar únicamente las recepciones asociadas al gateway propio utilizado en el proyecto.
